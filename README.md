# Verbiverse

**Verbiverse** 定位是使用 LLM 来辅助外语 PDF 阅读工具，用以提升语言能力。

## Usage

- Pre-build 详见 Rlease 页面

- 源码运行：
  1. 使用 poetry 安装依赖：`poetry install`
  2. 运行程序：`poetry run python main.py`
